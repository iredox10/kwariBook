# KwariBook

Specialized Bookkeeping for traders in Kantin Kwari, Kano.

## Tech Stack
- **Frontend:** React + Vite + Tailwind CSS (v4)
- **PWA:** vite-plugin-pwa
- **Database:** Dexie.js (Offline-first)
- **Backend:** Appwrite
- **Localization:** i18next (English & Hausa)

## Getting Started

### Prerequisites
- [Bun](https://bun.sh) installed

### Installation
```bash
bun install
```

### Development
```bash
bun dev
```

### Build
```bash
bun run build
```
